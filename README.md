# Enterprise Infrastructure Administration Lab

## Overview

This repository documents my hands-on IT Administration lab, built to polish and strengthen the skills I already have by simulating a more complete enterprise-style infrastructure environment.

The lab started with core on-premises infrastructure using Windows Server and Active Directory, then expanded into hybrid identity, Microsoft 365 administration, Exchange Online, and SharePoint Online.

The goal of this project is to imitate how different systems work together in a real enterprise environment, including:

- identity and access management
- policy enforcement
- storage and permissions management
- redundancy and continuity
- operating system deployment
- patch management
- hybrid identity synchronization
- Microsoft 365 administration
- department-based communication
- SharePoint-based operational processes

---

## Project Goals

- Repolish and strengthen practical IT Administration skills
- Simulate a real enterprise-style infrastructure environment
- Combine on-premises and cloud administration in one lab
- Document the setup clearly through screenshots, notes, and lab guides
- Showcase hands-on skills across system administration, hybrid identity, Microsoft 365, Exchange, and SharePoint

---

## Environment Used

### Hardware
- Dell PowerEdge R430 enterprise server

### Virtualization
- Proxmox

### On-Premises Technologies
- Windows Server
- Active Directory Domain Services
- Group Policy
- File Server Resource Manager (FSRM)
- NTFS and share permissions
- Data Deduplication
- Windows Server Update Services (WSUS)
- Windows Deployment Services (WDS)
- Secondary Domain Controller for redundancy

### Cloud Technologies
- Microsoft 365 Admin Center (E5)
- Microsoft Entra ID
- Microsoft Entra Connect Sync
- Exchange Online
- SharePoint Online
